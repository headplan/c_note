# 快速总结

#### 基本语法

**固定格式**

```c
#include <stdio.h>

int main() {
    printf("Hello, World!\n");
    return 0;
}
```

**printf语句**

```c
#include <stdio.h>

int main() {
    printf("Hello, World!\n");
    printf("你好,世界!\n");
    return 0;
}
```

> \n - 换行 . \t - 空格
